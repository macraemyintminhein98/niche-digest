# NicheDigest

NicheDigest provides a curated, ad-free digest of headlines and links from highly specific RSS feeds. Get the high-value content you need without the information overload.

## Features

- **Ad-Free Reading**: Focus solely on content.
- **Customizable Feeds**: Add your niche RSS sources.
- **Clean Digest**: Easy-to-read, summarized updates.

## Technologies (Planned)

- **Frontend**: Next.js/React
- **Backend**: Python (Flask/FastAPI) or Node.js (Express) for scraping (Vercel Functions/Render)
- **Database**: Firebase Firestore (free tier)
- **Payments**: Stripe Checkout

## How to use (MVP)

1. Input your niche RSS feed URLs.
2. Receive a clean digest of the latest articles.
3. Subscribe for unlimited feeds and premium features.
